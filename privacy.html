# Privacy Policy - Ciel Finance Tool

Last updated: April 2026

## 1. Data Collection
This tool only accesses data necessary 
for content posting functionality.

## 2. Data Usage
Collected data is used solely for 
posting content to the authorized 
TikTok account.

## 3. Data Sharing
No user data is shared with third parties.

## 4. Data Storage
Authentication tokens are stored locally 
and securely on the developer's machine.

## 5. Contact
For privacy concerns, contact via 
TikTok developer portal.
